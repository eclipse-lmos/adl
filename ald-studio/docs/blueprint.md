# **App Name**: Promptitude

## Core Features:

- Prompt Editor: A markdown editor for crafting and modifying prompts.
- Test Case Management: Display and manage a list of test conversations. Select one to see the full history.
- Execution Engine: Run prompts against selected models and manage settings such as token count or model temperature.
- Performance Analysis: Provide visualization and comparison of the different prompt performances.
- AI Prompt Tool: AI-powered tool to optimize prompts based on test results. This feature will evaluate a user prompt, execute a battery of tests, then present the findings and reasoning about any suggested changes to improve the prompts using specific LLM 'tool' requests.

## Style Guidelines:

- Primary color: A vibrant blue (#2E9AFE) to convey clarity and focus.
- Background color: Light gray (#F5F7FA) for a clean, neutral canvas.
- Accent color: A saturated purple (#9B51E0) to draw attention to key elements such as the 'Run' button.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text, because its clean, neutral appearance will minimize distraction and allow the prompts and test results to be the focus of attention.
- A responsive three-column layout for editing prompts, managing tests, and displaying performance results. Each column should adapt to different screen sizes.
- Simple, consistent icons for actions like 'Run,' 'Edit,' and 'Save' to enhance usability.
- Subtle transitions and animations to provide feedback on interactions, like a loading animation while tests are running.
